## 코드 블록 주의 사항

메인 문서에는 언어가 `python`으로 지정된 코드 블록이 118개 있습니다. 이 숫자는 실행 파일이나 자동 테스트 수가 아니라 설명·예제·정답을 포함한 Markdown 코드 블록 수입니다.

다음 세 위치의 블록에는 `return redirect(...)`가 들어 있으며 독립 Python 프로그램이 아닙니다.

- 문서 도입부의 Flask CRUD 흐름 예시

이 조각들은 Flask 라우트 함수 내부라는 전제와 `request`, `redirect`, `messages` 등의 정의가 필요합니다. 문서 안의 다른 짧은 `request.form.get(...)` 참고 조각도 Flask 요청 문맥 없이는 실행할 수 없습니다.

콘솔 예제를 실습할 때는 설명용 Flask 조각을 제외하고, 필요한 초기 리스트·딕셔너리와 `input()`이 함께 있는 하나의 완결된 예제 또는 정답 블록을 선택하세요.
